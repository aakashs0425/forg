# Water Wise

## Sip by sip, on track.

Water Wise is an AI-powered hydration tracking web application that helps users monitor their daily water intake, track hydration goals, view analytics, and maintain healthy hydration habits.

## Features

* User Registration and Login
* Water Intake Tracking
* Hydration Dashboard
* Daily Progress Monitoring
* Analytics and Charts
* AI Hydration Assistant
* Weather-Based Hydration Suggestions
* Responsive Design

## Core Screens

1. Landing Page
2. Login Page
3. Register Page
4. Dashboard
5. Water Log Page
6. Analytics Page
7. AI Assistant Page

## Technology Stack

### Frontend

* React.js
* Vite
* Chart.js

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

## Run Locally

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```
